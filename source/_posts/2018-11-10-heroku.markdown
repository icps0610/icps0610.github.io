### reset base
``heroku restart; heroku pg:reset DATABASE --confirm `cat .git/config | grep heroku.com | awk -F [/.] '{print $6}'` && heroku run rake db:migrate``


### rename app
`git remote rm heroku`  
``heroku git:remote -a [new_app_name]`
