#### informatrion
linux版本 : `lsb_release -a`    
cpu : `cat /proc/cpuinfo`  
cpu temp : `vcgencmd measure_temp`  
mem : `cat /proc/meminfo`   
記憶體使用狀況 : `free -m`   
硬體資訊 : `dmidecode`  
顯示卡型號 : `lspci | grep VGA`  
硬碟資訊 :  `smartctl -a /dev/sda`  
USB :  `lsusb`  
kernel版本 : `uname -a`  
dmesg : `kernel logs`
